# MovieLens Analytics

Анализ датасета MovieLens с помощью объектно-ориентированного подхода.  
Проект включает:
- Загрузку и обработку файлов `ratings.csv`, `tags.csv`, `movies.csv`, `links.csv`.
- Вычисление статистик по фильмам, пользователям, тегам.
- Парсинг дополнительной информации с IMDB (бюджет, сборы, режиссёр, хронометраж).
- Модульные тесты для проверки всех методов.
